# 5-Graduation-Projects-Part-1
تهدف مبادرة "انشاء مشروع تخرجك بنفسك" إلى توفير موارد تعليمية شاملة تساعد الطلاب على إنشاء وتنفيذ مشاريع تخرجهم بشكل مستقل ومتكامل. تتألف هذه المبادرة من عدة أجزاء، يحتوي كل جزء منها على 5 مشاريع تخرج كاملة، تتضمن الجوانب العملية والنظرية. سيتم رفع هذه المشاريع على منصة GitHub لتكون مرجعاً متاحاً لجميع الطلاب.
